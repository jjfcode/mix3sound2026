## 🎨 Customization

### Changing Brand Colors
Edit `css/style.css` and modify the CSS variables:

```css
:root {
    --primary-color: #FF4B00;    /* Main brand color */
    --accent-color: #00D4FF;     /* Accent color */
    --text-dark: #1a1a1a;        /* Dark text */
    /* ... more variables ... */
}
```

### Adding Images
1. Create `images/` folder if it doesn't exist
2. Organize images into subfolders (hero, portfolio, services, team)
3. Optimize images before adding (use TinyPNG.com)
4. Replace placeholder divs with `<img>` tags

## 🔧 Deployment

### Option 1: GitHub Pages (Free)
1. Push code to GitHub
2. Go to repository Settings → Pages
3. Select branch to deploy
4. Your site will be live at `https://[username].github.io/mix3sound2026`

### Option 2: Netlify (Free)
1. Sign up at [Netlify.com](https://www.netlify.com)
2. Connect your GitHub repository
3. Deploy with one click
4. Custom domain supported

### Option 3: Traditional Web Hosting
1. Upload all files to your web hosting via FTP
2. Ensure folder structure is maintained
3. Access via your domain name
