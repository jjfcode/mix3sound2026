MIX3 SOUND LOGO FILES
=====================

Place your logo files in this folder:

REQUIRED FILE:
- mix3-logo.png      Main logo for navigation (transparent background recommended)
                     Recommended size: 200px wide x 50-60px tall
                     Format: PNG with transparency
                     Color: Can be #515050 or white, depending on your design

OPTIONAL FILES:
- mix3-logo-white.png    White version for dark backgrounds (footer)
- mix3-logo-dark.png     Dark version for light backgrounds
- mix3-icon.png          Square icon version for favicon (32x32px or 64x64px)

TIPS:
- Use PNG format for best quality with transparency
- Keep file sizes under 100KB
- Test the logo visibility on both light and dark backgrounds
- Logo should be readable at small sizes (mobile devices)

The logo is currently used in:
- Navigation bar (all pages)
- Can also be added to footer if desired

Current setup uses: images/logo/mix3-logo.png
